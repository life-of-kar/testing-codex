## overusage_notice_post.sh
To enable custom disk quota notifications in DirectAdmin:

1. Copy `overusage_notice_post.sh` into `/usr/local/directadmin/scripts/custom/`.
2. Set executable permissions: `chmod 755 /usr/local/directadmin/scripts/custom/overusage_notice_post.sh`.
3. DirectAdmin will automatically invoke this script when a user exceeds their disk quota.
